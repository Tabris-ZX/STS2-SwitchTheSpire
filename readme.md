# <div align="center">SwitchTheSpire</div>

<div align="center">支持调换游戏中卡牌/遗物/药水的获取. 可将三者中的任意一种转换为另一种, 商店直购不受影响.</div>

## 功能

- **三向互换**: 卡牌 ⇄ 遗物 ⇄ 药水, 自由配置每种奖励的转换目标
- **F7 设置面板**: 游戏中按 F7 弹出设置框,保存后即刻生效

## 使用

游戏内按 **F7** 打开设置面板, 分别设置卡牌/遗物/药水各转换为哪种奖励, 点击保存即刻生效.

配置文件位于模组目录下的 `config.json`, 也可手动编辑.
